Create `utils/` as a Python package with four modules. All code must be production-quality with type hints and docstrings.

**utils/logging_config.py:**
- Function `setup_logging(name: str, level: int = logging.INFO) -> logging.Logger`
- Configures a logger with format: `"%(asctime)s | %(levelname)-8s | %(name)s | %(message)s"`
- Date format: `"%H:%M:%S"`
- Returns configured logger instance
- Per D-04: tutorials use both `logging` (for technique output) and `print()` (for section headers)

**utils/benchmark.py:**
- Decorator `benchmark(func)` — wraps a function, measures wall-clock time (using `time.perf_counter`) and optionally GPU memory (using `torch.cuda.max_memory_allocated` if CUDA available). Returns a dict `{"result": func_return, "time_seconds": float, "memory_mb": float_or_none}`.
- Function `compare_results(baseline: dict, optimized: dict, technique_name: str) -> None` — prints a formatted comparison table per D-05 showing:
  ```
  ┌─────────────────────────────────────────────────────┐
  │ Benchmark Results: {technique_name}                  │
  ├──────────────┬──────────────┬──────────────┬────────┤
  │ Metric       │ Baseline     │ Optimized    │ Change │
  ├──────────────┼──────────────┼──────────────┼────────┤
  │ Time (s)     │ {baseline}   │ {optimized}  │ {x}x   │
  │ Memory (MB)  │ {baseline}   │ {optimized}  │ {diff} │
  └──────────────┴──────────────┴──────────────┴────────┘
  ```
- Function `print_benchmark_table(results: list[dict]) -> None` — prints multiple benchmark rows for tutorials that compare more than two configurations.

**utils/models.py:**
- Class `SimpleCNN(nn.Module)` — 3 conv layers + 2 FC layers, accepts `input_channels=3, num_classes=10, input_size=32` params. Forward returns logits.
- Class `SimpleMLP(nn.Module)` — 3 linear layers with ReLU, accepts `input_dim=784, hidden_dim=256, output_dim=10`.
- Function `get_sample_batch(batch_size=32, channels=3, height=32, width=32, device=None) -> tuple[torch.Tensor, torch.Tensor]` — returns random input tensor and random integer labels.

**utils/device.py:**
- Function `get_device() -> torch.device` — returns CUDA if available, else CPU. Logs which device selected.
- Function `print_device_info() -> None` — prints GPU name, memory, CUDA version if CUDA available; else prints CPU info.

**utils/__init__.py:**
```python
from utils.logging_config import setup_logging
from utils.benchmark import benchmark, compare_results, print_benchmark_table

    "setup_logging",
    "benchmark", "compare_results", "print_benchmark_table",

    "get_device", "print_device_info",
]
```
